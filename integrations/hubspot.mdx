## How to connect your HubSpot account

1. Navigate to the [Apps & Integrations](https://app.novacal.io/integrations) page and go to [HubSpot](https://app.novacal.io/integrations/hubspot)
2. Click the **Connect** button and complete the HubSpot authorization flow
3. After the connection is complete, Novacal will be ready to sync contacts from the event types you enable for HubSpot

## How to choose which event types sync to HubSpot

You can control HubSpot syncing per event type in two places:

### From the HubSpot integration page

1. Navigate to [HubSpot](https://app.novacal.io/integrations/hubspot)
2. In the **Sync Event Types** section, turn the switch **on** for each event type you want to sync
3. Turn the switch **off** for any event type you want to exclude from HubSpot syncing

### From the event type editor

1. Go to the [Event Types](https://app.novacal.io) page and open the event type you want to manage
2. Navigate to the **Apps** tab
3. Find **HubSpot** and use the switch to enable or disable syncing for that event type

## What gets synced

When HubSpot is connected and enabled for an event type, Novacal syncs booked contacts into HubSpot by:

- Matching an existing HubSpot contact by email when possible
- Creating the contact when it does not already exist

## How to disconnect HubSpot

1. Navigate to [Apps & Integrations](https://app.novacal.io/integrations) page and then go to [HubSpot](https://app.novacal.io/integrations/hubspot)
2. Click the **Disconnect** button
