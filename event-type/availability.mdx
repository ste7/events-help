## Viewing Availability

This page displays the availability profile currently assigned to this event type. You can see the weekly schedule, and all custom availability overrides for specific dates.

## Selecting Availability

You can select a different availability profile from your existing profiles. Each profile can have its own timezone and schedule, allowing you to assign different availability rules to different event types.
